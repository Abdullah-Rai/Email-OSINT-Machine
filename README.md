# Advanced Email OSINT Machine

A Python-based OSINT tool that performs email reputation analysis, DNS OSINT, breach lookup, Gravatar lookup, SPF/MX/DMARC checks, and more.

## Features
- Email format validation
- Domain analysis
- HIBP breach lookup
- Gravatar lookup
- DNS OSINT (A, MX, SPF, DMARC)
- Human-readable summary & JSON output
